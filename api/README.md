## Technologies
* Node 12 LTS
* NestJS
* PostgreSQL
* TypeORM
* Typescript
* GraphQL
## Intructions to start the project.
* execute `npm install`
* create .env file with
    ```
  NODE_ENV=develop
  SENDGRID_KEY=
  S3_REGION=
  S3_SECRET_KEY=
  S3_PUBLIC_KEY=
  S3_BUCKET=
  AUTH_PRIVATE_KEY=
  AUTH_PUBLIC_KEY=
  DB_USER=
  DB_HOST=
  DB_PASSWORD=
  DB_NAME=
  PORT=3000
  DASHBOARD_URL=
  LANDING_URL=
  ```
* execute 
 `npm run start:dev`
* let's go!
